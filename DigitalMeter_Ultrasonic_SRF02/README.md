## Digital Meter using SRF02 Ultrasonic Moudle
	   
MCU:		PIC16F877A   
Frequency:     	8MHz (External Crystal)   
Display:        16x2 LCD   
Sensor:		SRF02 Ultrasonic Moudle (I2C Interfacing)     

Note: Included schematic and PCB layout with Proteus
